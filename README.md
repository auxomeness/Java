# Java Practice and OOP Exercises Collection

This repository contains a collection of Java programs created for learning and practicing core programming concepts, particularly Object-Oriented Programming (OOP). The files include exercises, lab activities, and sample implementations covering fundamental to intermediate Java topics.

---

## Overview

The purpose of this repository is to:

* Build a strong foundation in Java programming
* Practice Object-Oriented Programming principles
* Explore core programming constructs such as loops, classes, and inheritance
* Serve as a reference for academic exercises and coding practice

## Contents

### Basic Programs

* **SampleAddition.java** – Simple arithmetic operations
* **loops.java** – Looping constructs and iterations
* **sample.java** – General practice code

### Object-Oriented Programming

* **ClassesObjects.java** – Introduction to classes and objects
* **GetterExample.java** – Encapsulation using getters
* **InheritancePractice.java** – Basic inheritance concepts
* **SampleInheritance.java** – Demonstration of inheritance

### Advanced OOP Concepts

* **Pavia_MyLittlePoly.java** – Polymorphism examples
* **Pavia.java / IntroPAVIA.java** – Course-related exercises

### Lab Exercises

* **LAB1_.java**
* **LAB1_2_Pavia.java**

### Miscellaneous

* **Austin.java**, **Trial.java**, **Main.java** – General or experimental programs

* Java Development Kit (JDK) 8 or higher

## Compilation

To compile a Java file:

```bash id="r8v3l1"
javac FileName.java
```

Example:

```bash id="x2f9p0"
javac Main.java
```

---

## Running Programs

```bash id="k1m4z9"
java ClassName
```

Example:

```bash id="q7n2s8"
java Main
```

## Notes

* Each `.java` file is a standalone program with its own `main()` method (in most cases)
* Files are independent and are not intended to be compiled together unless specified
* Some files may be experimental or incomplete
